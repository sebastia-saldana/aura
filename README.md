# aura
Prueba de conocimiento para AURA

Entrar en la carpeta server con:
 - cd server/


Correr lo migrate con:
 - adonis migration run
 
 
Llevar la base de datos con:
 - adonis seed
 
 
Correr el servidor con:
 - adonis serve -dev


Carmbiar a carpeta de cliente con:
 - cd ..
 - cd client/
 
 
Correr cliente con:
 - npm start
